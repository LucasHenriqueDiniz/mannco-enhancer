# Changelog

All notable changes to this project will be documented in this file.

## [0.1.1] - 2026-04-27
### Added
- **UI/UX Improvements**: Refined popup layout and added custom scrollbar styling.
- **Asset Cleanup**: Removed unused provider logos to reduce extension size.
- **Documentation**: Fully elaborated `README.md` and `PRIVACY_POLICY.md`.
- **Localization**: Enhanced translation strings for Portuguese, Spanish, and Russian.

### Fixed
- **Accessibility**: Fixed ARIA spam in profile pages to reduce browser lag.
- **UI**: Corrected label spacing and alignment in the language selector.

## [0.1.0] - 2026-04-20
### Added
- Initial Mannco Enhancer starter base using Manifest V3.
- Popup-driven settings flow with typed storage synchronization.
- Background service worker and content script infrastructure.
- Core feature set: Inventory highlights, Quick Update modal, and Buy-order safety tools.
- Support for multiple locales (EN, PT-BR, ES, RU).
